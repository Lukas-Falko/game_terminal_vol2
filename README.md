# Podstawowe komendy Git

## Inicjalizacja i klonowanie
- `git init` – inicjalizacja nowego repozytorium Git
- `git clone <URL>` – sklonowanie istniejącego repozytorium

## Status i zmiany
- `git status` – sprawdzenie statusu plików w repozytorium
- `git add <FILE>` – dodanie pliku do obszaru staging
- `git add .` – dodanie wszystkich zmian do obszaru staging
- `git commit -m "komentarz"` – zapisanie zmian z komentarzem
- `git diff` – pokazanie różnic między wersjami plików
- `git log` – wyświetlenie historii commitów

## Gałęzie
- `git branch` – lista gałęzi
- `git branch <NAZWA>` – utworzenie nowej gałęzi
- `git checkout <NAZWA>` – przełączenie na inną gałąź
- `git merge <NAZWA>` – scalanie gałęzi z aktualną

## Repozytorium zdalne
- `git remote -v` – wyświetlenie powiązanych repozytoriów zdalnych
- `git pull` – pobranie i scalenie zmian z repozytorium zdalnego
- `git push` – wysłanie lokalnych commitów do repozytorium zdalnego

## Inne przydatne
- `git rm <FILE>` – usunięcie pliku z repozytorium
- `git stash` – tymczasowe odłożenie zmian
